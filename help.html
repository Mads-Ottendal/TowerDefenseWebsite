HOW TO PLAY THE GAME

To play this game you first have to download the game by clicking the link from the game tab

You have downloaded the game but to play it you need to have Java on your computer. This is because the game is a java file.

To check if you have Java on your computer open the 'Terminal' on your computer and write: java -version

If you already have Java click here

If you do not have Java click the link here to go to Java website to download Java

You then have to pick the correct version of Java for you computer. If you have a Mac you need to figure out if it has an Intel CPU or a Apple Silicon CPU (m1/m2).

The way you figure which chip you Mac has, is by clicking the apple icon in the top left corner

You then click About This Mac

You then click And look at the chip

You then click You then pick the version of Java that is compatible with your chip

When you have Java the only thing left is to open the game but since it is downloaded from the internet then you cant just double click as you will be met with a warning

You just have to right-click the file and click on open

You will then be met with a new pop-up that asks if you're sure you want to open it. Here you just have to click open

When that is done the game should open, and you should be met with the game menu screen
